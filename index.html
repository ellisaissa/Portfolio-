  • BarU

    This was the final project done for the bootcamp that I attended. It was done with a group. While the initial idea was mine I was happy to see how the idea evolved and changed as I worked with my group.

    The idea is that you can look up a drink based on the alcohol used in it. This way it makes it easier when you want to try something new but don't want to spend a lot of time looking through a menu or having the bartender list drinks.

    With this project I was on the front-end part of the team. I pair programmed to make the general layout, color scheme, and results pages. I also created the loading page. Click to here see deployed project

  • DinGen

    This was the second project done for the bootcamp that I attended. The idea came about after discussing how hard it is to figure out what you want to eat when you're already hungry.

    The name of the project is short for dinner generator. You can type in an ingredient and see options for meals you can cook, along with recipes. But if you don't find anything that looks good, there's a tacos button. Because, really, who doesn't like tacos?

    On this project I worked on the front-end. I helped create the modals and color scheme. I also made the social media for the project and connected the icons. Click to here see deployed project
